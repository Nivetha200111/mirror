# The Mirror

Success compatibility engine that matches an Indian origin story to a mentor using 5D vectors, surfaces the biggest gap, and drafts an AI bridge.

## Stack
- Next.js 14 (App Router) + TypeScript
- Tailwind CSS + Framer Motion
- OpenAI (GPT-4o-mini) for advice

## Run locally
```bash
npm install
npm run dev
```

## Environment
Set `OPENAI_API_KEY` to enable live bridge advice (see `.env.example`).
